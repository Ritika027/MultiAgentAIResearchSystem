# Multi-Agent AI Research System 🤖

## 🚀 Overview

This project is a **Multi-Agent AI Research System** built using **LangChain, Python, and Large Language Models (LLMs)**. It automates the process of researching any topic by using multiple AI agents that work together in a pipeline.

Instead of using a single AI model, this system divides the task into specialized agents that handle different parts of the workflow such as searching, reading content, writing reports, and reviewing results.

---

## ⚙️ How It Works

The system follows a simple pipeline:

1. **User Input** – Enter a research topic
2. **Search Agent** – Finds relevant information from the web
3. **Reader Agent** – Extracts and processes useful content
4. **Writer Agent** – Generates a structured research report
5. **Critic Agent** – Reviews and improves the final output
